What Can JavaScript Do?

JavaScript can change HTML attributes.

In this case, JavaScript changes the SRC (source) attribute of an image using the "getElementById()" command.


JavaScript can change the style of HTML elements.


JavaScript can hide HTML elements by using the "getElementById()" command to change the "style.display" attribute to "none".



JavaScript can show hidden HTML elements by using the "getElementById()" command to change the "style.display" attribute to "block".
